# คู่มือการใช้งานระบบยืมอุปกรณ์

## ภาพรวมระบบ

## บทบาทผู้ใช้

### Admin (ผู้ดูแลระบบ)
- จัดการทั้งหมด: อุปกรณ์, ผู้ใช้, รายงาน
- อนุมัติคำร้อง: ตรวจสอบและอนุมัติการยืม
- ดูสถิติ: รายงานและสถิติการใช้งาน
- จัดการผู้ใช้: เพิ่ม, แก้ไข, ลบสมาชิก
- ถังขยะ: จัดการข้อมูลที่ถูกลบ

### Member (ผู้ใช้ทั่วไป)
- ยืมอุปกรณ์: ส่งคำร้องยืม
- ดูประวัติ: ประวัติการยืมของคุณ
- จัดการโปรไฟล์: แก้ไขข้อมูลส่วนตัว
- สแกน QR: รับอุปกรณ์ด้วย QR Code

---

## การยืมอุปกรณ์

### ค้นหาอุปกรณ์
1. ใช้ช่องค้นหา:
   - พิมพ์ชื่ออุปกรณ์
   - ระบบจะแสดงผลลัพธ์ทันที

2. กรองตามหมวดหมู่:
   - เลือกหมวดหมู่ที่ต้องการ
   - ดูอุปกรณ์ในหมวดหมู่นั้น

### ส่งคำร้องยืม
1. ค้นหาอุปกรณ์ที่ต้องการ
2. คลิกที่อุปกรณ์เพื่อดูรายละเอียด
3. คลิก "ยืมอุปกรณ์"
4. กรอกข้อมูล:
   - วันที่ต้องการยืม: เลือกวันที่
   - จำนวน: ระบุจำนวนที่ต้องการ
   - หมายเหตุ: บอกวัตถุประสงค์การยืม
5. คลิก "ส่งคำร้อง"

### ติดตามสถานะคำร้อง
- รออนุมัติ: รอ admin ตรวจสอบ
- อนุมัติแล้ว: พร้อมรับอุปกรณ์
- ถูกปฏิเสธ: คำร้องไม่ผ่านการอนุมัติ
- กำลังยืม: กำลังใช้งานอยู่
- คืนแล้ว: ส่งคืนเรียบร้อย

---

## ระบบ QR Code

### สร้าง QR Code สำหรับรับอุปกรณ์
1. ไปที่หน้าประวัติการยืม
2. หาคำร้องที่อนุมัติแล้ว
3. คลิก "แสดง QR Code"
4. รอ QR Code ปรากฏ
5. แคปเจอร์ QR Code ด้วยมือถือ

### สแกน QR Code สำหรับรับอุปกรณ์ (Admin)
1. ไปที่หน้าสแกน QR Code
2. อนุญาตการเข้าถึงกล้อง
3. สแกน QR Code ที่ผู้ใช้แสดง
4. ระบบจะยืนยันการรับอุปกรณ์
5. อุปกรณ์พร้อมให้ผู้ใช้นำไปใช้

### ความปลอดภัยของ QR Code
- หมดอายุ: QR Code หมดอายุใน 5 นาที
- การเข้ารหัส: มี checksum ป้องกันการปลอม
- ใช้ครั้งเดียว: แต่ละ QR Code ใช้ได้ครั้งเดียว
- ตำแหน่ง: จำเป็นต้องอยู่ในสถานที่ที่กำหนด

---

## การดูประวัติ

### ประวัติการยืม
1. ไปที่หน้าแดชบอร์ด
2. ดูส่วน "ประวัติการยืม"
3. กรองตาม:
   - ช่วงวันที่: เลือกช่วงเวลา
   - สถานะ: กำลังยืม, คืนแล้ว
   - ค้นหา: หาอุปกรณ์ที่ต้องการ

### ข้อมูลที่แสดง
- ชื่ออุปกรณ์: รายละเอียดอุปกรณ์
- วันที่ยืม: วันที่เริ่มยืม
- วันที่คืน: วันที่ส่งคืน
- สถานะ: สถานะปัจจุบัน
- QR Code: ปุ่มสร้าง QR Code (ถ้าอนุมัติ)

### ส่งออกรายงาน
1. ไปที่หน้ารายงาน (Admin)
2. เลือกประเภทรายงาน
3. กำหนดช่วงวันที่
4. คลิก "ส่งออก CSV"
5. บันทึกไฟล์ CSV ลงเครื่อง

---

## การจัดการสำหรับ Admin

### จัดการอุปกรณ์
1. ไปที่เมนู "จัดการอุปกรณ์"
2. เพิ่มอุปกรณ์ใหม่:
   - กรอกข้อมูลทั้งหมด
   - อัปโหลดรูปภาพ
   - กำหนดจำนวน
3. แก้ไขอุปกรณ์:
   - คลิกปุ่มดินสอง
   - แก้ไขข้อมูล
   - บันทึกการเปลี่ยนแปลง
4. ลบอุปกรณ์:
   - ตรวจสอบว่าไม่มีคนยืม
   - คลิกปุ่มถังขยะ

### จัดการผู้ใช้
1. ไปที่เมนู "จัดการสมาชิก"
2. เพิ่มผู้ใช้ใหม่:
   - กรอกข้อมูลส่วนตัว
   - อัปโหลดรูปโปรไฟล์
   - กำหนดสิทธิ์ (Admin/Member)
3. แก้ไขข้อมูล:
   - แก้ไขข้อมูลส่วนตัว
   - เปลี่ยนรูปโปรไฟล์
   - ปรับสิทธิ์
4. ลบผู้ใช้:
   - ไม่สามารถลบตัวเองได้
   - ตรวจสอบก่อนลบ
2. **เพิ่มผู้ใช้**:
   - คลิกปุ่ม "เพิ่มผู้ใช้"
   - เลือกบทบาท (user/admin)
   - กรอกข้อมูลผู้ใช้
3. **แก้ไขผู้ใช้**: คลิกไอคอนแก้ไขข้อมูล
4. **ลบผู้ใช้**: คลิกปุ่ม "ลบ" เพื่อลบผู้ใช้

### 6. ดูสถิติระบบ
1. ไปที่เมนู "สถิติระบบ"
2. **รายงานที่มี**:
   - สถิติการยืมทั้งหมด
   - อุปกรณ์ที่ถูกยืมมากที่สุด
   - ผู้ใช้ที่ยืมมากที่สุด
   - สถิติตามช่วงเวลา

## ขั้นตอนการยืมอุปกรณ์ (สำหรับนักศึกษา)

### ขั้นที่ 1: ค้นหาอุปกรณ์
1. ล็อกอินเข้าระบบ
2. ดูรายการอุปกรณ์ที่มีให้
3. ตรวจสอบจำนวนคงเหลือ

### ขั้นที่ 2: ส่งคำร้องยืม
1. เลือกอุปกรณ์ที่ต้องการ
2. กรอกจำนวนที่ต้องการยืม
3. เลือกวันที่ต้องการรับอุปกรณ์
4. เพิ่มลงตะกร้า

### ขั้นที่ 3: ยืนยันคำร้อง
1. ตรวจสอบรายการในตะกร้า
2. คลิก "ยืนยันการยืม"
3. ระบบจะส่งคำร้องให้ admin พิจารณา

### ขั้นที่ 4: รอการอนุมัติ
1. รอ admin ตรวจสอบและอนุมัติคำร้อง
2. หากอนุมัติ: ระบบจะแจ้งเตือน
3. หากปฏิเสธ: ระบบจะแจ้งเตือนพร้อมเหตุผล

### ขั้นที่ 5: รับอุปกรณ์
1. หลังได้รับการอนุมัติ
2. ไปรับอุปกรณ์ตามวันที่กำหนด
3. ใช้งานตามปกติ

### ขั้นที่ 6: คืนอุปกรณ์
1. ไปที่ "ประวัติการยืม"
2. ค้นหารายการที่ต้องการคืน
3. คลิกปุ่มคืนอุปกรณ์
4. ส่งคืนอุปกรณ์คืน

---

## ข้อควรพึงระวม

### สำหรับนักศึกษา
- ต้องรอการอนุมัติจาก admin ก่อนสามารถรับอุปกรณ์
- ต้องคืนอุปกรณ์ภายในเวลาที่กำหนด
- สามารถยืมได้ครั้งละต่อครั้งตามจำนวนที่คงเหลือ
- ต้องรักษาอุปกรณ์ให้สภาพเป็นปกติ

### สำหรับผู้ดูแลระบบ
- ตรวจสอบคำร้องยืมทุกวัน
- อนุมัติหรือปฏิเสธอย่างเหตุผลที่ชัดเจน
- ตรวจสอบจำนวนอุปกรณ์ก่อนอนุมัติ
- จัดการข้อมูลผู้ใช้และอุปกรณ์ให้เป็นปัจจุบัน

---

## ปัญหาและวิธีแก้ไข

### ปัญหาที่พบบั่วย
1. **ไม่สามารถล็อกอินได้**
   - ตรวจสอบ username และ password
   - ติดต่อผู้ดูแลระบบ

2. **ไม่พบอุปกรณ์ที่ต้องการ**
   - อุปกรณ์อาจะถูกยืมหมดแล้ว
   - ตรวจสอบสถานะอุปกรณ์ในหน้าจัดการอุปกรณ์

3. **ส่งคำร้องไม่ได้**
   - ตรวจสอบว่ากรอกข้อมูลครบถ้วย
   - ตรวจสอบว่าอุปกรณ์มีให้ยืม

4. **ไม่สามารถคืนอุปกรณ์**
   - ตรวจสอบว่าอุปกรณ์อยู่ในสถานะ "กำลังยืม"
   - ติดต่อผู้ดูแลระบบ

---
